# react-strapi-game-store
E-commerce completo de jogos digitais, com sistema de pagamento, área do cliente e CMS customizado. Os usuários podem buscar, filtrar, adicionar jogos ao carrinho e finalizar a compra. Administradores têm acesso a um painel para gerenciar produtos, categorias, promoções e conteúdos do site, com envio automático de e-mails de venda.
